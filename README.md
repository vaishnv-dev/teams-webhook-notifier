# Microsoft Teams Webhook Notifier

## Overview
This project sends automated messages to Microsoft Teams using Incoming Webhooks.

## Features
- Real-time notifications
- Simple API integration
- Lightweight and easy to use

## Tech Stack
- Python
- REST API
- Microsoft Teams

## How to Run
1. Create a Teams webhook
2. Replace webhook URL in app.py
3. Run:
   python app.py

## Use Case
Used for sending alerts and notifications to Teams channels.
